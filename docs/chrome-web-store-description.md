## Full Description

### Transform Your Browsing Experience

Brutalist Chrome is a premium browser customization that combines a bespoke dark monochrome Chrome theme with a feature-rich Manifest V3 extension. Inspired by brutalist architecture — raw structure, honest materials, and purposeful design — this extension transforms Chrome into a focused, efficient workspace.

The carefully crafted color palette centers on deep blacks and charcoals (#111111, #1a1a1a, #2a2a2a) with warm gold accents (#c4a882) that add a touch of refinement. Every visual element has been designed to reduce eye strain, minimize distractions, and create a cohesive browsing environment.

### What Makes It Different

Brutalist Chrome is not just a theme — it's a complete browser enhancement toolkit. While other themes simply change colors, Brutalist Chrome adds a full suite of productivity tools accessible from a beautifully designed popup, sidebar, and command palette.

### Key Features

| Feature | Description |
|---------|-------------|
| **Dark Monochrome Theme** | Full Chrome theme with custom colors for toolbar, tabs, bookmarks bar, new tab page, and all Chrome UI surfaces. Warm gold accents provide subtle visual hierarchy. |
| **Command Palette** | Press Ctrl+Shift+P to open a fuzzy-search command palette. Instantly access every feature without navigating menus. Type "bookmark github" and jump directly to your GitHub bookmarks. |
| **Smart Bookmarks Manager** | Search, browse, and open bookmarks from anywhere. Supports folder navigation, fuzzy search by title and URL, and one-click opening. |
| **Tab Manager** | View all open tabs across windows. Search, switch to, or close tabs without hunting through crowded tab bars. |
| **History Viewer** | Browse and search your browsing history. Filter by date range, search by keywords, and revisit pages instantly. |
| **Session Manager** | Save groups of tabs as named sessions. Restore them later with one click. Perfect for project contexts, research sessions, or daily workflows. |
| **Clipboard Manager** | Never lose a copied item again. The clipboard manager stores your last 50 copied texts. Browse history and click to re-copy any item. |
| **Todo Manager** | Quick-add todos with Ctrl+Shift+Q. Manage your task list from the popup or sidebar. Todos persist across browser restarts and sync automatically. |
| **Quick Notes** | Jot down ideas without leaving your current page. Sticky notes with customizable colors that persist across sessions. |
| **Weather Widget** | Live weather on your new tab page. Configure your city and OpenWeatherMap API key for real-time conditions, temperature, and forecast icons. |
| **Focus Mode** | Block distracting websites with a configurable block list. Toggle focus mode on and off to stay productive. |
| **Context Search** | Select any text on any page, press Ctrl+Shift+F, and instantly search it. No copy-paste needed. |
| **Keyboard Shortcuts** | All features have customizable keyboard shortcuts. Set them to whatever feels natural. |
| **Sidebar Panel** | Toggle the sidebar (Ctrl+Shift+S) for a persistent workspace that stays open while you browse. |
| **Customizable** | Change accent colors, set custom background images, inject custom CSS, and remap every keyboard shortcut from the options page. |

### Privacy Promise

Brutalist Chrome is built on a privacy-first philosophy:
- **Zero data collection** — We do not collect, track, or transmit any personal data
- **Local storage only** — All settings, todos, notes, and clipboard history stay in your browser
- **No analytics** — No tracking scripts, no telemetry, no third-party services (except weather, which you configure)
- **Open source** — The complete source code is available on GitHub for audit

### Weather Feature

The optional weather widget requires a free API key from OpenWeatherMap. You enter your own key — we never see it. Weather data is cached locally and refreshed every 30 minutes.
